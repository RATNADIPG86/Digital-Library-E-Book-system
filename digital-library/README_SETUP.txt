SETUP STEPS
1. Copy the folder to C:/xampp/htdocs/
2. Start Apache and MySQL from XAMPP.
3. Open http://localhost:8080/phpmyadmin or http://localhost/phpmyadmin depending on your Apache port.
4. Create/import database using database.sql.
5. Open the website:
   http://localhost:8080/digital-library/Index.php
   or
   http://localhost/digital-library/Index.php
